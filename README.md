# Chroniques Astrales

Idle game en francais inspire des classiques du genre, avec une direction artistique moderne.

## Lancer le projet

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Features V1

- Ressource principale: essence astrale
- Production active et passive
- Systeme d'achats (ouvriers + rituel)
- Sauvegarde automatique locale
- Gain hors-ligne (max 8h)
